db Module¶

db.createDB(path=None, startyear=2010, futureyear=False)[source]¶

Used to create the RPL and tax databases. If the path is blank, the location of the script is the location of the database. startyear will start with 2010 unleses otherwise specified by using a four digit year as an integer. If futureyear is set to True it will create records for the next rollyear (i.e. if it is 2012 it will create 2013.)

db.updateDB(path=None, startyear=2010, futureyear=False, update=False)[source]¶

Used to update the RPL and tax databases. If the path is blank the location of the script is the location of the database. startyear will begin with 2010 unleses otherwise specified by using a four digit year as an integer. If futureyear is set to True it will create records for the next rollyear (i.e. if it is 2012 it will create 2013.) If update is set to True, a temporary database will be used.
