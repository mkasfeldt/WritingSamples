makegis Module¶

makegis.makeGISfile(*args, **kwds)[source]¶

Creates a file for the GIS system, then return GIS data in a dictionary. rollyear is the four digit year of the roll year you wish to export into the tab.gis file, if this is not set it will default to the current year. taxyear is the four digit year that will be used to get the tax data from the property database (current year taxes, previous year taxes, lottery credit count and lottery credit), if it is not set it will be the same year as the roll year. dbpath is the path to the property.sqlite database, if it is not set it will be the path to the script running this function. filepath is the path where the gis.tab file will be created.
