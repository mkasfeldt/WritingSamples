makelrip Module¶

makelrip.makeLRIPfile(*args, **kwds)[source]¶

Create the lrip.tab file which contains property information to be used by the general public and return data. rollyear is the four digit year of the roll year you wish to export into the lrip.tab file, if it is not set it will default to the current year. dbpath is the path to the property.sqlite database, if it is not set it will be the location of the script running this function. filepath is the path where the lrip.tab file will be created.
