Creating an Item Requirement from Inventory in iFinance

Group: D365 - iFinance - How To's - iFinance (AP Traffic Cop)

Modified on: 2023-07-26 @ 14:47:18Z

Pupose: Create an item requirement from inventory for a project that require Traffic Cop corrective actions.


Prerequisites:

  • Access to D365 iFinance
  • The project ID
  • The part's MPR number


To create an item requirement from inventory:

1. Open iFinance and login.

2. Navigate to Project management and accounting > Projects > All projects.

Figure 1 - Navigate to All Projects


3. Click the Project ID link of the required project.

  • Note: Use filtering to reduce the result set.

Figure 2 - Click the Project ID Link


4. Click the Plan item from the top menu, then select the Item requirements from the ribbon.

Figure 3 - Select Item Requirements


5. Click the +New button, then enter the required information in the following fields:

  • Item Status = Inventory
  • Item number = The item's MPR number
  • Quantity = The quantity needed of the part
  • Warehouse = The warehouse where the part is located
  • Location = The location in the warehouse where the part is located

Figure 4 - Enter the Item Requirement's Information


6. Click the Save button in the top menu.

Figure 5 - Click the Save Button


Outcome: The item requirement has been added to the project from inventory.
